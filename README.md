# TDS Page Resources

A sleek resource downloader for Tower Defense Simulator and ALTER EGO wiki documentation.

Everything inside the `public` folder is under CC-BY-SA 3.0.

## Build

### Install

```bash
bun i
```

### Development

```bash
bun run dev
```

### Build

```bash
bun run build
```

Output directory: `dist`

### Preview

```bash
bun run preview
```
